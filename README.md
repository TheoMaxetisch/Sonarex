# Sonarex

Swift/iOS-App für Musik, Playlists und persönliche Bibliothek.

## Projektstruktur

- `Sonarex/` enthält den App-Code
- `SonarexTests/` enthält Unit Tests
- `Sonarex.xcodeproj/` enthält das Xcode-Projekt
- `Unterlagen/` enthält Projektunterlagen

## Setup

1. Repository klonen
2. `Sonarex.xcodeproj` in Xcode öffnen
3. App starten
